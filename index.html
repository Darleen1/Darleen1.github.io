封面

写给离开我们的人，写给陪伴我们的人，写给每个人心中的山与海。

  • 刘十三，一个洒脱的人，他的小本本上记满了要完成的事，最遥远的是考上清华北大，因为他妈妈告诉他要好好学习，去大城市生活。还有卖保险与情敌打赌，一年要卖一千零一份，不为别的，就为争口气。最后外婆走了，一人踉踉跄跄爬到山顶为外婆点灯笼，照亮外婆回家的路。
  • 程霜，真的是非常酷且优秀的女孩，从小就知道自己的病活不了多久，却还一直乐观，因而一直有奇迹发生。记忆最深则是，每次离开，都会给刘十三一个承诺，如果能活着回来，就做他女朋友，最后一次是和他结婚，然后结果不尽人意，最后仍被病魔带走了生命。程霜一直爱着刘十三，两人在喝酒划拳时就特别让我心疼她，但她自己也很矛盾，知道自己生命有限，会连累刘十三，但又讨厌别人欺负他，只有自己能欺负他，小时候是，长大以后还是。所以他和女朋友分手时，连夜租车带他去南京找前女友说个清楚，那句刘十三加油，给了他很大勇气。青春就是疯狂的。
  • 外婆王莺莺，对外孙细腻且爱与包容，以及他们之间相处的和谐与美好，都让人羡慕。可能年少轻狂，可能还不懂失去的痛苦，这也是男主为什么叫刘十三，失散的原因。外婆的小卖部一直在，外婆也想等妈妈回来，外婆的强大底下也会有脆弱的一面。外婆一个人一天一夜开着拖拉机把醉醺醺的刘十三从城市里拉回来，一直像小时候一样，给他帮助，和他斗嘴，打骂，都是外婆对他无私的爱。外婆做的饭肯定非常好吃，外婆的小卖部可能赚不了多少钱，但外婆就一直经营，因为刘十三说考不上大学，就来帮她。可刘十三考上大学了，而且最终外婆没能逃离死亡的爪印。外婆生前拖人买了录音笔，其实就是录遗言，她希望刘十三可以好好的，她也是个母亲，她也希望女儿能回来看她一眼，陪她喝喝小酒，聊聊天，顿时一个坚强的人，内心的柔软就表现出来了，让人心痛不已。
  • 还有球球，一个可爱且懂事的小孩，爸爸妈妈因为别人钱家里债不换疯的疯，死的死，爸爸疯，确想到为球球交学费，球球知道爸爸最爱吃鸡蛋糕，偷偷准备好准备带回去给爸爸吃。爸爸知道球球开学要交学费了，就去银行取钱，可他哪来的钱，终究都是一些债条，银行工作人员不理他，他确挥着斧头砍人家，误伤了她。然后爸爸就悲剧的被警察枪毙了。

最近所学的git命令

初始化一个Git仓库，使用git init命令。
添加文件到Git仓库，分两步：
使用命令git add ，
使用命令git commit -m “”
git differ用来查看修改了什么
git status查看仓库状态
git log显示从最近到最远的提交日志
git reset可以把暂存区的修改退回到工作区
git reset –hard commit-id回退到以前的任意版本
git reflog查看命令历史
二次修改文件需要git add 两次 再git commit
Cat命令用于查看当前文件的内容
git checkout –撤销文件修改
git checkout 切换分支
rm删除工作区文件
git rm 删除版本库文件并且git commit -m “”
git remote add origin git@GitHub 关联仓库
git push -u origin master 把本地仓库所有内容推送到远程仓库，并关联分支
git push origin master 推送master分支最新的修改
git clone git账号 ：克隆本地仓库
git checkout -b issue-…
git stash apply恢复，stash内容并不删除，用git stash drop来删除；
git stash pop，恢复的同时把stash内容也删了。
rebase操作可以把本地未push的分叉提交历史整理成直线；
git branch -D强行删除分支
git pull把最新的提交从origin/dev抓下来，用于与小伙伴的最新提交和你试图推送的提交有冲突，然后，在本地合并，解决冲突，再推送
git tag 用于新建一个标签，默认为HEAD，也可以指定一个commit id；
git tag -a -m “blablabla…”可以指定标签信息；
git tag可以查看所有标签。
git push origin ..推送一个本地的标签
git push origin –tags推送全部未推送的本地标签
git tag -d ..删除本地标签
git push origin :refs/tags/..删除远程标签
